Use this to add a Aufgabe.

  • aufgabenstellung – String to define the aufgabenstellung of Aufgabe

  • loesung – String to define the loesung of Aufgabe

  • schwierigkeit – int to define the schwierigkeit of Aufgabe

  • zeit – int to define the zeit of Aufgabe

Returns

0 on success, -1 on failure
